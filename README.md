


# **README – Experiment 11: Data Analysis using Python (NumPy & Pandas)**

## **Aim**

To perform data analysis using Python by applying NumPy and Pandas functions for data manipulation, statistical computation, and extracting meaningful insights from structured datasets.

---

## **Theory**

Data analysis involves inspecting, cleaning, transforming, and modeling data in order to discover useful information and support decision-making. In this experiment, Python libraries such as **NumPy** and **Pandas** are used to efficiently handle numerical and tabular data.

**NumPy** is primarily used for numerical computations and working with arrays, while **Pandas** provides powerful data structures like DataFrames to store and manipulate structured data. Together, they enable fast processing, easy data handling, and efficient analysis.

The process of data analysis typically includes:

* Loading and understanding the dataset
* Cleaning and organizing the data
* Performing statistical operations
* Extracting patterns and summarizing results

### **Functions Used and Their Uses**

#### **NumPy Functions**

* **`import numpy as np`**
  Imports the NumPy library.
  *Use:* Provides support for large, multi-dimensional arrays and mathematical operations.

* **`np.array()`**
  Creates an array.
  *Use:* To store numerical data in a structured format.

* **`np.mean()`**
  Calculates the average of elements.
  *Use:* To find central tendency of data.

* **`np.median()`**
  Finds the median value.
  *Use:* To determine the middle value of a dataset.

* **`np.std()`**
  Computes standard deviation.
  *Use:* To measure spread or variability in data.

* **`np.sum()`**
  Calculates the sum of elements.
  *Use:* To find total values.

* **`np.max()` and `np.min()`**
  Find maximum and minimum values.
  *Use:* To identify range limits.

---

#### **Pandas Functions**

* **`import pandas as pd`**
  Imports the Pandas library.
  *Use:* Provides tools for data manipulation and analysis.

* **`pd.read_csv()`**
  Loads dataset from a CSV file.
  *Use:* To import data into a DataFrame.

* **`pd.DataFrame()`**
  Creates a DataFrame.
  *Use:* To organize data into rows and columns.

* **`head()` and `tail()`**
  Display first and last few rows.
  *Use:* To preview dataset.

* **`info()`**
  Provides summary of dataset structure.
  *Use:* To check data types and missing values.

* **`describe()`**
  Generates statistical summary.
  *Use:* To analyze distribution of numerical data.

* **`shape`**
  Returns dimensions of dataset.
  *Use:* To understand size of data.

* **`columns`**
  Lists column names.
  *Use:* To identify variables.

* **`isnull()` and `sum()`**
  Detect missing values.
  *Use:* To clean and preprocess data.

* **Indexing (`df['column']`)**
  Access specific column.
  *Use:* To work with individual variables.

* **Filtering (`df[condition]`)**
  Select rows based on condition.
  *Use:* To extract relevant data.

* **`sort_values()`**
  Sorts data.
  *Use:* To arrange values in order.

---
